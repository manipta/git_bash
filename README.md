# Git Bash
## For Linux
dos2unix command: converts a DOS text file to UNIX format.
```
dos2unix git.sh
./git.sh
```

## For Windows/DOS
Unix2dos command: converts a Unix text file to DOS format
```
unix2dos git.sh
./git.sh
```
